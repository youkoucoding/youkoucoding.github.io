HTML

<p>和<br>的区别

  • <p>块级元素，<br> 内联元素；
  • <p>能被 css 修改，<br> 不能；
  • <p>非单标签元素，<br> 是单标签元素；
  • <p>换行靠的是块级元素特性，<br> 换行靠的可能是类似 \n 的渲染规则

TypeScript

  • Transform a union to another union, using the in operator as a kind of for-loop
type Entity =
  | {
      type: 'user';
    }
  | {
      type: 'post';
    }
  | {
      type: 'comment';
    };

// to implement below:

type EntityWithId =
  | {
      type: 'user';
      userId: sting;
    }
  | {
      type: 'post';
      postId: string;
    }
  | {
      type: 'comment';
      commentId: string;
    };
  • solution:
type EntityWithId = {
  [EntityType in Entity['type']]: {
    type: EntityType;
  } & Record<`${EntityType}Id`, string>;
}[Entity['type']];

const result: EntityWithId = {
  type: 'comment',
  commentId: '12345',
};
