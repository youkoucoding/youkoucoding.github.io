JS

var person = {
	age:1
}
Object.defineProperty(person,'age',{
	get(){
		return this.age
	},
	set(val){
		this.age = val
	}
})

person.age   Error：Maximum call stack size exceeded，循环引用导致栈溢出
  • person.age --> get.call(person) -->this.age --> person.age --> get.call(person) --> this.age -->...
