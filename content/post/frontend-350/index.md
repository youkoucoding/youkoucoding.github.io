## JavaScript

### `Math.ceil()`, `Math.round()`, `Math.floor()`

- `Math.ceil()` 函数返回大于或等于一个给定数字的最小整数

```js
console.log(Math.ceil(0.95));
// expected output: 1

console.log(Math.ceil(4));
// expected output: 4

console.log(Math.ceil(7.004));
// expected output: 8

console.log(Math.ceil(-7.004));
// expected output: -7
```

- `Math.round()` 函数返回一个数字四舍五入后最接近的整数

- `Math.floor()` 返回小于或等于一个给定数字的最大整数
